# Download Pose Models (Important)

## 1. Download COCO model
Download from:
https://www.dropbox.com/s/2h2bv29a130sgrk/pose_iter_440000.caffemodel

Then copy the file to:
`pose/coco`


## 2. Download MPI model
Download from:
https://www.dropbox.com/s/drumc6dzllfed16/pose_iter_160000.caffemodel

Then copy the file to:
`pose/mpi`  


Make sure the files are placed correctly before running the code.
